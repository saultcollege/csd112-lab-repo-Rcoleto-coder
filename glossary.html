Glossary

HTML
HyperText Markup Language - It is used to mark structural and semantic meaning in documents
CSS
Cascading Style Sheet - It is a rule based language that gives that structural and semantic markup an actual appearance
JavaScript
It is a programming language that can be used to add interactivity to a website
Hyperlink
Hyperlinks allow us to link documents to other documents or resources, link to specific parts of documents, or make apps available at a web address
Unordered lists
Unordered lists are used to mark up lists of items for which the order of the items doesn't matter
